Project One Website

This website was created by Andres Felipe Lopez Lozano using only HTML.

Purpose

The purpose of this site is to demonstrate clean HTML structure across multiple pages, using correct navigation, media, and validation. The project includes a resume page and a topic page focused on Saint Seiya.

What is included

  • A homepage that introduces the site
  • A resume page with professional information
  • A topic page about Saint Seiya
  • Images and an audio clip as required media

Site overview

Each page includes a navigation list at the top. All links and media use directory-based paths inside this repository.
